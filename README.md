# Password Generator using Python

This project generates random passwords based on user preferences.
It is a beginner-friendly program to understand randomization and strings.

---

## About the project

The user can choose:
- Password length
- Whether to include uppercase letters
- Numbers
- Symbols

The program then generates a random password based on these choices.
